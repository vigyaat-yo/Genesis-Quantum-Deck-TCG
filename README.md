# ⚛️ Genesis: Quantum Deck – TCG

> Command machines.  
> Control stats.  
> Conquer the arena.

**Genesis: Quantum Deck** is a fast-paced **anime-style mech trading card game** where you deploy powerful combat robots, compare tactical stats, and outsmart your opponent in intense 1v1 battles.

Build your deck.  
Upgrade your cores.  
Dominate the battlefield.

---

## 🎮 Gameplay Overview

Each player receives **15 cards**.

Every round:
1. Both players reveal their top card
2. Current player selects a stat
3. Higher value wins
4. Winner takes both cards
5. Winner chooses next stat
6. First player to collect **all 30 cards** wins

No searching.  
No cheating.  
Pure strategy + luck.

---

## ⚙️ Card Stats

Each card contains 4 attributes:

| Stat | Unit | Description |
|---------|---------|---------------------------|
| Power | HP | Raw combat strength |
| Processor | THz | AI calculation speed |
| Velocity | km/h | Movement & attack speed |
| Memory | TB | Tactical capacity |

👉 **Higher value always wins**
